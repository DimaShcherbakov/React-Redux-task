Вверху страницы шапка, на которой есть лого (https://image.ibb.co/k7cmVT/logo_w.png ) 
и две ссылки - home link (ведет к перезагрузке страницы) и Pixelplex link 
(открывает https://pixelplex.io в новой вкладке)

Task 1. Страница делится на две части: слева - input, 
в который мы вводим текст, и кнопка, которая добавляет введенный текст в список, 
который находится справа. Справа в списке у каждой строки находится кнопка,
 которая удаляет эту строку из списка. При выполнении данного задания добавление текста в список,
  а так же его удаление должно происходить с помощью принципов Redux.

Task 2. Страница делится на 2 части:
 слева должен быть любой Range Slider с интервалом от 1 до 5000 и шагом 100, 
 справа - счетчик миллисекунд, шаг изменения которого зависит от значения, 
 выбранного в Slider. Настройка шага должна сохраняться и не должна 
 сбрасываться после перезагрузки страницы, счетчик должен начинаться с 
 нуля после каждой перезагрузки страницы. Для хранение параметра шага 
 используйте Redux.

Результат выполнения тестового задания разместите на аккаунте Github, 
а для демонстрации реализации воспользуйтесь функционалом Github Pages.
